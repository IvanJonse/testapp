## Description
Basic dependences

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

nodemailer - using for send a message to the mail
swagger - documentation
bcrypt - hash password
typeorm - ORM (for connecting to DB)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
